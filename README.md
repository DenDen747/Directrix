# Directrix

## Commands

 - ``login <username> <password>``
   - Logs in to the server
   - ``<status>``
 - ``signup <username> <password>``
   - Registers an account
   - ``<status>``
 - ``list conversations``
   - Lists all conversations
   - ``<status> [conversations]``
 - ``list messages <username>``
   - Lists all messages in a conversation
   - ``<status> [messages]``
